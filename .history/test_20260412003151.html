My Book

L'application qui vous permet de prendre vos notes, stocker vos mots de passe, vos cours organisés en chapitres ou sections et ces sections en contenus unitaires, ainsi que vos médias, et de synchroniser le tout sans les perdre entre votre mobile et votre PC via le cloud.

Pourquoi la version Android est plus volumineuse ?

Notre application fonctionne sur tous vos appareils grâce à une technologie moderne. Voici pourquoi la taille diffère :

  • Version Android (68 Mo) : Contient tout le nécessaire pour fonctionner sur n'importe quel téléphone Android, même sans connexion. C'est votre compagnon mobile complet.
  • Version Windows (36 Mo) : S'appuie sur les outils déjà présents sur votre PC pour être plus légère. Idéale pour votre bureau.

Les deux versions offrent exactement les mêmes fonctionnalités et synchronisation cloud.
