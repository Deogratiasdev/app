My Book

L'application qui vous permet de prendre vos notes, stocker vos mots de passe, vos cours organisés en chapitres ou sections et ces sections en contenus unitaires, ainsi que vos médias, et de synchroniser le tout sans les perdre entre votre mobile et votre PC via le cloud.

Attention — Si ce site n'est pas https://mybookmobilepc.pages.dev, veuillez ne rien télécharger.

Pourquoi l'APK est plus lourd ?

L'APK Android est plus volumineux car il doit inclure tout ce dont l'application a besoin, alors que Windows utilise les composants système déjà présents :

  • Moteur Flutter runtime (~20 MB) - Android doit l'inclure, Windows l'optimise
  • Icônes FontAwesome (~4 MB) - Inclues dans l'APK, Windows les partage
  • Polices système (~2 MB) - Android les inclut, Windows utilise celles du système
  • Assets et ressources (~5 MB)
  • Bibliothèques natives Android (~10 MB) - Windows utilise les bibliothèques système
  • Code Dart compilé (~5 MB)
